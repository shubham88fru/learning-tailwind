# learning-tailwind

# 1. Colors classes.

```css
.text-yellow-400 {
} /*font color to yellow*/

.bg-yellow-500 {
} /*background color to yellow*/
```

# 2. Text styles

```css
.text-center {
} /* center the text.*/

.text-xl {
} /* large text.*/

.text-[120px] {
} /* font size of 120px*/

.font-semibold {
} /*font weight to 600*/

.uppercase {
} /* make the text uppercase*/

.tracking-[.35em] {
} /* for letter spacing of .35em*/
```
